Analiza Factorial (AF)

Analiza Factorial este o tehnic statistic utilizat pentru a identifica structura ascuns a datelor. Ea pornete de la premisa c variaiile variabilelor observate (cele 13 specificaii tehnice) sunt determinate de un numr mai mic de factori lateni (nevzui).

  • Scopul principal este de a descoperi "cauzelor profunde" care explic de ce anumite variabile sunt corelate ntre ele.
  • De exemplu, variabilele observate "Vitez maxim", "Acceleraie 0-100" i "Putere CP" sunt puternic corelate deoarece n spatele lor st un singur factor latent: "Performana Sportiv". AF ne ajut s gsim i s cuantificm acest factor.
  • Spre deosebire de ACP (care doar comprim datele matematic), Analiza Factorial modeleaz relaiile de cauzalitate dintre concepte abstracte i msuri concrete.

Teste de compatibilitate

  1. Testul Bartlett de Sfericitate:
    • Verific ipoteza nul c variabilele sunt necorelate (matricea de corelaie este identitate).
    • Un p-value < 0.05 (semnificativ statistic) ne indic faptul c putem respinge ipoteza nul. Exist corelaii semnificative ntre variabile, deci analiza factorial are sens.
In[4]:
chi2,p_value=calculate_bartlett_sphericity(x)
print(f'Chi patrat: {chi2}\nP-value: {p_value}')
if p_value<0.05:
    print('H(0): variabilele sunt independente (RESPINS)')
    print('H(1): variabilele nu sunt independente (ACCEPT)')
else:
    print('H(0): variabilele sunt independente (ACCEPT)')
    print('H(1): variabilele nu sunt independente (RESPINS)')
    print('NU se poate analiza factoriala')
    sys.exit(-10)

Gasirea numarului de factori latenti

Folosind Criteriul Kaiser (valori proprii > 1), determinm ci factori lateni sunt necesari pentru a explica structura datelor.

  • O valoare proprie (eigenvalue) reprezint cantitatea de informaie coninut ntr-un factor. Dac valoarea este > 1, nseamn c acel factor explic mai mult variaie dect o singur variabil original, deci merit pstrat.
  • Codul ne indic numrul optim de factori pe care s i extragem pentru modelul final.

Comunalitati si varianta specifica

  • Comunaliti: Reprezint procentul din variaia fiecrei variabile (ex: "Consum", "Putere") care este explicat de factorii comuni de baz. Valori apropiate de 1 indic o reprezentare bun n model.
  • Varian Specific: Este partea de informaie care rmne unic variabilei sau eroare de msurare (neexplicat de factori). Este complementul comunalitii ($1 - h^2_j$). Valorile mici sunt ideale.
