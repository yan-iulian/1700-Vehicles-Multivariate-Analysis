Crearea modelului de clusteri ierarhici

Reprezint structura matematic a dendrogramei, stocnd ordinea n care mainile sunt unite n clusteri. Fiecare rnd descrie o etap de unire: care dou entiti (sau sub-clusteri) sunt comasate i la ce distan.

Metoda Ward utilizat aici minimizeaz variaia n interiorul clusterilor la fiecare pas, formnd grupuri ct mai compacte i omogene din punct de vedere statistic.

Cautm numrul optim de clusteri

Pentru a evita alegerea direct a numrului de clusteri, calculm diferenele dintre distanele de agregare succesive. Punctul n care apare cel mai mare "salt" (diferen maxim) indic pragul natural de separare.

  • Acesta este momentul n care unirea a doi clusteri ar fora mpreun grupuri prea diferite, crescnd brusc heterogenitatea.
  • Codul identific acest punct (k_max) i sugereaz numrul ideal de clase (n acest caz, 2 clusteri).
In[5]:
k_max=np.argmax(h[1:,2]-h[:-1,2])
nr_clusteri=p-k_max
print(f"Numarul optim de clusteri este: {nr_clusteri}")
#creare partitie automata cu fcluster
auto=fcluster(h,nr_clusteri,criterion='maxclust')

Indeci Sillhouette

Msoar calitatea segmentrii pentru fiecare main n parte, cuantificnd ct de bine se potrivete un obiect n clusterul su comparativ cu celelalte. Valoarea variaz ntre -1 i 1:

  • Valori spre 1: Maina este "tipic" pentru clusterul su i foarte diferit de restul (0.5+ indic o structur clar).
  • Valori spre 0: Maina este la grania dintre dou categorii, avnd caracteristici hibride (nu este decisiv plasat).
  • Media scorurilor: Indic validitatea global a partiiei (cu ct mai mare, cu att segmentarea este mai robust).

Crearea funciei de desenare histogram

Aceast funcie construiete grafice comparative pentru o variabil aleas (variabila), afind cte o histogram pentru fiecare cluster n parte.

Rolul ei este de a evidenia diferenele de distribuie dintre grupuri (de exemplu, dac un cluster conine doar maini cu Power mare, iar altul doar cu Power mic).

In[11]:
def histograma(x,variabila,partitie):
    clustere_unice=np.unique(partitie)
    fig,axs=plt.subplots(1,len(clustere_unice),figsize=(10,6),sharey=True)
    fig.suptitle(f"Histograme ale variabilei {variabila}")
    for ax,cluster in zip(axs,clustere_unice):
        ax.hist(x[partitie==cluster],bins=15,rwidth=0.9)
        ax.set_title(cluster)

  • Clusterul 1: Prezint o coad lung spre dreapta (valori pozitive mari). Acesta grupeaz mainile puternice, cu motoare mari (supercar-uri sau SUV-uri de lux). Valorile sunt dispersate, indicnd o varietate de motoare de top.
  • Clusterul 2: Distribuia este concentrat n jurul mediei (valori apropiate de 0 sau negative, dup standardizare). Aici se afl mainile obinuite de serie, cu motorizri standard, mult mai omogene.
